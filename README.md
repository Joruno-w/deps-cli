A CLI tool to manage dependencies in your project

## Install
```bash
npm install -g deps-cli
```

## Usage
```bash
deps-cli
```

## Configuration
```ini
// ~/deps-cli.ini
[auth]
username=xxx
password=xxx
cookies="xxx"

[projects]
root=xxx

[presets]
data={}
```
